# dkb-web

Web corporativa de dinkbit España (`dinkbit.es`). Stack: Next.js 16 · TypeScript · Tailwind v4. Spec en [docs/MASTERPLAN.md](./docs/MASTERPLAN.md).

## Comandos

```bash
npm run dev          # http://localhost:3000
npm run build
npm run typecheck
npm test             # vitest
```

## Variables de entorno

Ver `.env.example`.
